Informações gerais do Projeto:
Aplicação foi desenvolvida para postar imagem, pensameto, ideias, reportagem e o que o usuário achar de importante. Para o primeiro acesso deve se cadastrar na plataforma, informando nome, e-mail e senha, no caso o e-mail é único não podendo ser repetido.
Após o cadastro o usuário é direcionado para começar a postar, ou se preferir pode mudar a senha de acesso. Outra opção é a pesquisa por data de postagem.

Instalação:
